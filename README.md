# FactGuard: AI Verification System (Hackathon Project)

Autonomous multi-agent constellation verifying truth at the speed of light.

## Project Overview
FactGuard is a real-time AI verification dashboard designed to debunk misinformation and verify claims using a multi-agent system (Planner, Executor, Evaluator).

## Features
- **Real-time Claim Analysis**: Verify text, URLs, or documents.
- **Multi-Agent Workflow**: Visualized agent reasoning process.
- **Live Monitoring**: System status, global misinformation levels, and news ticker.
- **Cyberpunk UI**: A futuristic, high-contrast interface.

## Tech Stack
- React 19 + Vite
- TailwindCSS
- Lucide React Icons

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
